# 📈 Crypto EMA Trend Bot

Алгоритмический торговый бот для криптовалют (Bybit) на основе
**EMA-trend стратегии**.

Бот поддерживает: 
- 📊 Бэктест стратегии 
- 🔬 Мульти-бэктест по нескольким монетам 
- 🧪 Paper trading (виртуальная торговля)
- 🤖 Live trading (реальная торговля)
- 📩 Telegram уведомления

Стратегия ориентирована на **trend-following** и использует **EMA-exit
вместо фиксированного тейк-профита**, что позволяет ловить большие
тренды.

------------------------------------------------------------------------

# ⚙️ Стратегия

## Основная идея

Бот ищет **начало тренда** и удерживает позицию пока тренд продолжается.

### Вход LONG

- EMA fast пересекает EMA slow вверх
- RSI ниже лимита
- EMA spread достаточно большой
- EMA slope положительный
- Цена выше slow EMA
- HTF тренд подтверждает направление

### Вход SHORT

- EMA fast пересекает EMA slow вниз
- RSI выше лимита
- EMA slope отрицательный
- Цена ниже slow EMA
- HTF тренд подтверждает направление

------------------------------------------------------------------------

# 📉 Выход из позиции

Используется **EMA-exit** вместо фиксированного тейк-профита.

LONG закрывается если:

    close < EMA_fast

SHORT закрывается если:

    close > EMA_fast

Также используется **фиксированный Stop Loss (обычно 2%)**.

Это позволяет удерживать длинные тренды и получать большие прибыльные
сделки.

------------------------------------------------------------------------

# 📊 Режимы работы

## Backtest

Тест стратегии на исторических данных:

    python bot.py --mode backtest

------------------------------------------------------------------------

## Multi Backtest

Массовый тест сразу по нескольким символам:

    python multi_backtest.py

Проверяются:

- BTCUSDT
- ETHUSDT
- SOLUSDT
- XRPUSDT
- TRXUSDT

И два временных окна:

- last_90d
- prev_90d

------------------------------------------------------------------------

## Paper Trading

Виртуальная торговля без реальных денег.

    python bot.py --mode paper

Используются реальные рыночные данные, но сделки выполняются виртуально.

------------------------------------------------------------------------

## Live Trading

Реальная торговля:

    python bot.py --mode live

⚠️ Использовать только после длительного тестирования.

------------------------------------------------------------------------

# 💰 Управление риском

Основные параметры:

- Stop Loss: **2%**
- Размер позиции: **10% от баланса**

Эти настройки находятся в `config.py`.

------------------------------------------------------------------------

# 🪙 Мульти-символьная торговля

Бот может торговать несколькими монетами одновременно.

Настраивается в `config.py`:

``` python
symbols = [
    "BTCUSDT",
    "ETHUSDT"
]
```

Добавление новой монеты:

``` python
symbols = [
    "BTCUSDT",
    "ETHUSDT",
    "TRXUSDT"
]
```

Каждый символ имеет: - отдельную позицию - общий баланс

# 🔧 Установка

## 1. Клонировать репозиторий

    git clone https://github.com/MikeMoore1337/crypto-ema-bot.git
    cd crypto-ema-bot

------------------------------------------------------------------------

## 2. Установить зависимости

    pip install -r requirements.txt

------------------------------------------------------------------------

## 3. Создать файл `.env`

    BYBIT_API_KEY=xxx
    BYBIT_API_SECRET=xxx

    TELEGRAM_BOT_TOKEN=xxx
    TELEGRAM_CHAT_ID=xxx

------------------------------------------------------------------------

# ⚙️ Основные настройки

В файле `config.py`.

### Таймфрейм

    interval = "5"

### Размер позиции

    position_size_pct = 0.10

### Стоп-лосс

    stop_loss_pct = 0.02

### Фильтры стратегии

    long_rsi_limit = 72
    short_rsi_limit = 35

    min_ema_spread_pct = 0.0006
    slope_lookback = 5

# 🧪 Рекомендуемый workflow

1. Запустить мульти-бэктест

```
    python multi_backtest.py
```

2. Выбрать стабильные символы

3. Запустить paper trading

```
    python bot.py --mode paper
```

4. Наблюдать минимум **2--4 недели**

5. После этого можно переходить в **live trading**.
